# Dicas comando top


 

## entrar no programa

$ top


* opções:

> `u` - username - para selecionar os processos de um usuário

> `<` - para mudar a coluna de ordenação

> `F` ou `O` - para ordenar os processos 
	
> `a` - para ordenar pelo pid

> `H` - para mostrar as threads/processos
